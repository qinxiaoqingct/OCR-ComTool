## OCR文本解析工具

OCR-ComTool 是基于Python QT5开发的一款带图形界面跨平台的串口工具，实现了接收OCR开发套串口指令并解析OCR文本功能。

## 工具使用
运行OCR文本解析工具：
- Windows：

双击comtool_Windows_v1.0.0.exe

- Linux&macOS：
```bash
cd comtool-Linux
chmod 777 comtool_Linux_v1.0.0
./comtool_Linux_v1.0.0
```

> 串口参数设置：
> 波特率:1500000
> 数据位:8
> 停止位:1
> 校验位:None
